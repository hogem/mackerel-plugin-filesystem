mackerel-plugin-filesystem
===

## Description

Get Linux FileSystem stats metrics for Mackerel.
This script can get nfs filesystem metrics.


## Usage

```
$ mackerel-plugin-filesystem
$ mackerel-plugin-filesystem -p /path_to_nfs_mount
$ mackerel-plugin-filesystem -p /hoge | mkr throw --service hoge
```


## Requirements

- Linux
- Python
